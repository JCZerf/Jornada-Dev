| Demo | [calculadora-imc-taupe-pi.vercel.app](https://calculadora-imc-taupe-pi.vercel.app) |

Calculadora de Índice de Massa Corporal com interface responsiva.

### Funcionalidades

- Cálculo do IMC (peso / altura²)
- Classificação automática em 8 categorias
- Validação de dados em tempo real
- Design responsivo

### Classificação IMC

| IMC | Classificação |
|-----|---------------|
| < 16 | Magreza Grau III |
| 16-16.9 | Magreza Grau II |
| 17-18.4 | Magreza Grau I |
| 18.5-24.9 | Adequado |
| 25-29.9 | Pré-Obeso |
| 30-34.9 | Obesidade Grau I |
| 35-39.9 | Obesidade Grau II |
| >= 40 | Obesidade Grau III |


- CSS Grid e Flexbox
- Validação de inputs numéricos
- Feedback visual por ranges
